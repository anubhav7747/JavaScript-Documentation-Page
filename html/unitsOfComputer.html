Units of Computer Memory
  • 1 Bit = Binary Digit
  • 8 Bits = 1 Byte
  • 1024 Byte = 1 KB
  • 1024 KB = 1 MB
  • 1024 MB = 1 GB
  • 1024 GB = 1 TB
  • 1024 TB = 1 Peta Byte
  • 1024 PB = 1 Exa Byte
  • 1024 EB = 1 Zetta Byte
  • 1024 ZB = 1 Yotta Byte
  • 1024 YB = 1 Bronto Byte
  • 1024 BB = 1 Geop Byte
